Navigation Flow

How the screens connect. Sheets are modal over any tab. "Start Navigation" is the only cross-tab jump.

Map Tab
Map · Idle
Start Trip button
→
Map · Recording
Live stats, seed drop, pause
→
Trip Summary
Bottom sheet, modal
Map · Idle
+ Route button
→
Route Builder
Tap waypoints, live routing
→
Name & Save
Saved to Explore tab
Explore Tab
Explore · Home
Trip list + route list + search
→
Trip Summary
Map, stats, elevation, seeds
Explore · Home
Tap a saved route
→
Route Detail
Elevation, stats, GPX export
→
Map · Navigate
Cross-tab jump, route loaded
Explore · Search
OSM trail query
→
Trail Preview
Mini map + stats
→
Save or Navigate
Save to routes or jump to Map
